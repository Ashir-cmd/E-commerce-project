E-commerce-project
A simple E-Commerce website built using HTML, CSS, and JavaScript.

The project fetches product data from the DummyJSON API and allows users to view product details, add products to the cart, and calculate the total bill.

## Features

- Display all products
- View product details
- Add products to cart
- Calculate total bill
- Store cart data using LocalStorage
- Fetch products from DummyJSON API

## Technologies Used

- HTML
- CSS
- JavaScript
- Fetch API
- Async/Await
- LocalStorage
